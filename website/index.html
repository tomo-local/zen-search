Features

🔍

Unified Search

Search across open tabs, bookmarks, history, and browser suggestions — all from one place.

⌨️

Keyboard First

Navigate results with arrow keys, open with Enter, and close with Escape. No mouse needed.

⚡

Lightweight & Fast

Minimal resource usage ensures smooth performance even with hundreds of tabs open.

🧮

Built-in Calculator

Type a math expression like 12 * 8 and get the answer instantly — no extra tab needed.
